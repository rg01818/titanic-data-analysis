# Titanic Data Analysis 

##  Project Overview
This project analyzes the Titanic dataset to understand survival patterns using Python.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis
- Survival rate by gender
- Age distribution
- Data cleaning (handling missing values)
- Visualization using charts

##  Files
- titanic notebook.ipynb

## Conclusion
Women had higher survival rates. Data cleaning and visualization helped in understanding patterns clearly.
